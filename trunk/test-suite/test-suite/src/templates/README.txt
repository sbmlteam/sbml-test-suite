
@file    src/templates/README.txt
@brief   Templates for files in the SBML Test Suite
@author  Michael Hucka

This directory contains templates for different kinds of files in the
SBML Test Suite.  Versions are provided for different programming
languages and situations.  When creating a new file to be included in
the Test Suite, (1) copy one of the templates and use it as the
starting point, (2) fill in the fields in the file that are
human-editable, such as the @file, @brief and @author lines near the
top.
