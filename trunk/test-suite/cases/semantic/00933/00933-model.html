The model contains one compartment called "C". There are two species named S1 and S2 and one parameter named k1. The model contains one reaction defined as:

Reaction Rate
S1 → S2 k1 · S1 · C

The model contains one event defined as:

Trigger Delay Assignments
Event1 S1 < 0.1 && S2 < 0.95 1 S1 = 1

The persistent attribute of the Trigger is set to 'true'. Thus, once the event has fired, even though the trigger value reverts to 'false' before the event is executed, execution must take place.

The initial conditions are as follows:

Value Units
Initial amount of S1 1 mole
Initial amount of S2 0 mole
Value of parameter k1 1 second-1
Volume of compartment "compartment" 1 litre

The species values are given as amounts of substance to make it easier to use the model in a discrete stochastic simulator, but (as per usual SBML principles) their symbols represent their values in concentration units where they appear in expressions.
