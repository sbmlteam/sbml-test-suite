The model contains one compartment called "C". There are three species named S1, S2 and S3 and one parameter named k1. The model contains one reaction defined as:

Reaction Rate
S1 → S2 k1 · S1 · C

The model contains one event that assigns value to species S3 defined as:

Trigger Delay Assignments
Event1 S1 ≤ 0.00005 && S1 ≥ 0.00004 3 S3 = 2.5 × 10−4

The persistent attribute of the Trigger is set to 'false'. Thus, once the event has fired, if the trigger value reverts to 'false' before the event is executed, then execution should not take place.
