SBML TEST SUITE - File Upload

Developed by Kimberly Begley
Version 0.1.0 beta 26 March 2008

Browse to a zipped file of results to be compared with known results. This upload accepts a zip file of comma delimited result files conforming to a naming standard where the file ends in the 5 digit test case name followed by csv, for example: myresultfile00001.csv
Files will be uploaded into a temporary directory on the server where they will be validated and compared to the known test results. A summary page with links to test details will be provided after the comparison is calculated.

Browse for zipped csv files:
e.g. C\:Program Files\SBML\MyModels\myresultfile00001.zip
