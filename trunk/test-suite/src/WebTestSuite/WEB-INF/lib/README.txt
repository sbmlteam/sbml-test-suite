
	       Explanation of this directory's purpose
			   SBML Test Suite

			    The SBML Team
			 http://www.sbml.org/
		     mailto:sbml-team@caltech.edu

This directory contains third-part libraries used by the Online SBML
Test Suite.  The files here get copied to the Tomcat webapps
installation directory on the server (http://sbml.org).

Copies of the original distribution files are in src/imported/.
Please see there for information about these libraries and their
licensing and distribution terms.
