SBML Testsuite
==============
This repository contains a re-write of the SBML Testsuite using Java, SWT and SWTCharts. 
This version is fashioned after the WIN32 version written before in .NET: 

http://sourceforge.net/projects/compare-results/

For more information see also the home page of the SBML Testsuite: 

http://sbml.org/Facilities/Online_SBML_Test_Suite
