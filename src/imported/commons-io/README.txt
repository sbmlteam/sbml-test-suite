
	       Explanation of this directory's purpose
		       src/imported/commons-io

			    Michael Hucka
			 http://www.sbml.org/
                  mailto:sbml-team@googlegroups.com

This directory contains the unmodified Apache Commons IO library,
obtained from http://commons.apache.org/io/.  The Commons IO library
provides utilities to assist with developing IO functionality useful
in Java-based web applications.  

The Online SBML Test Suite uses the Apache Commons IO library in its
implementation.  The file "commons-io-1.4.jar" is copied to the
directory /var/lib/tomcat5/webapps/test_suite/WEB-INF/lib/ in the
installation of the Online SBML Test Suite.

Apache Commons IO is Copyright 2001-2008 by The Apache Software
Foundation.  The Apache Commons IO licensing terms are described in
the file LICENSE.txt included in this directory.
